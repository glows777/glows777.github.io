# 垃圾回收机制

- 原始数据类型是存储在栈空间中的，引用类型的数据是存储在堆空间中的 => 解决了内存分配问题

## 回收方式

### 手动回收

- 如： C++
- 何时分配内存、何时销毁内存都是由代码控制的，这就容易造成内存泄漏等安全问题，但接近底层，灵活

### 自动回收

- 如Java，Python，JS
- 由垃圾回收器来回收，并不需要代码手动释放，减小心智负担
